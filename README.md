# Lighthouse Compass 7.4.2 — Multi-Theme Presenting & Visual Polish

## New
- Why Now? and What Would the Client Like to Be Different? now support multiple selections
- Large accordion headers are clearly labeled as main themes / umbrellas
- Each theme displays the number of selected subthemes
- Clinician guidance explains what each question is designed to elicit
- Prompts encourage identifying the one or two most impactful priorities
- Narrative generation supports multiple reasons for care and multiple desired outcomes
- Removed “From a bird’s-eye view” from chart-ready conceptualization language
- Selected checkboxes no longer lift, float, or leave a ghost outline
- Selected cards use a clean colored stripe, soft background, and color-coded domain accents
- Checkbox sizing and selected-state visibility were improved

All generated content requires clinician review and individualization.
No server-side PHI storage is included.
