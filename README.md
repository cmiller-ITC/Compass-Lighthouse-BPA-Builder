# Lighthouse Compass 6.5 — Diagnostic & Documentation Intelligence

## New in 6.5
- Automatic diagnostic rationale / justification
- Automatic medical-necessity narrative
- Automatic level-of-care rationale
- 5 Ps clinical formulation
- Initial treatment-focus recommendations
- Diagnostic confidence and clinical-status fields
- A dedicated Generate Diagnostic Narratives button
- Documentation packages for Assessment, Diagnosis & Formulation, and Risk & Safety
- Regenerate and Copy Package actions

All generated content is clinician decision support and must be reviewed and edited before use.

No database or server-side PHI storage is included.
